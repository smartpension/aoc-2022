## Prep day script

Use the `prep_day.sh` script to create the directory structure and pull the input file for the current day. If you wish to download an older day, specify that as `$1` eg..

```
./prep_day.sh ## Current day
./prep_day.sh 4 ## Create the directory for the 4th December
```
